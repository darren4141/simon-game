# simon-game
recreation of the simon says toy/game

## Components:
- Raspberry Pi Pico Microcontroller
- USB cable
- (assorted coloured LED) x 4
- Piezo speaker
- pushbutton x 4
- Jumper wires

## Features:
- start and end animations
- all of the features of the original game
